## 📋 Description

App Manager is a lightweight Windows application designed to help you manage, organize, and simplify the way you work with installed software. It provides a clean interface to view, launch, and control your applications without the bloat of the Windows default experience.

---

## 💻 System requirements

| Requirement | Minimum |
|---|---|
| OS | Windows 10 or Windows 11 (64-bit recommended) |
| RAM | 100 MB |
| Disk space | ~1 MB |
| Additional | None |

> Windows 7 / 8 may work but are not officially supported.

## 🚀 Installation

1. Download the latest release from the [Releases page](https://github.com/FarTekTV/app.manager/releases)
2. Run **`Setup.exe`**
3. Follow the on-screen instructions
4. Launch the app from the **Start Menu** or your **Desktop shortcut**

> **⚠️ Windows SmartScreen warning**
> If a SmartScreen popup appears, click **"More info"** then **"Run anyway"**.
> This is expected for apps without a recognized digital signature and is safe to dismiss.

---

## 🗑️ Uninstallation

You can uninstall App Manager through either:

**Settings** → Apps → Installed Apps → App Manager → Uninstall

or

**Control Panel** → Programs and Features → App Manager → Uninstall
